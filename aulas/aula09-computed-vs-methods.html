Comparação entre Computed properties e Methods

O resuktado final entre os dois será sempre o mesmo, porém, o computed é muito mais performático que o method, pois o vue cria um cache da variável computada que observa se a variável principal, no caso o text, foi alterada, já o method é chamado toda vez que uma alteração é feita e a cada vez que o conteúdo é renderizado.

Há também uma questão semântica, quando queremos uma ação, usamos o method, quando esperamos que a variável retorne alguma coisa, usamos o computed Ex: no campo script
