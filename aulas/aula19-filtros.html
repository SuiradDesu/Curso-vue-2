Filtros

Filtros / Filters, são responsáveis por formatar ou converter o conteúdo dos nossos dados/ variáveis.

Para criar um filter usamos a seguinte sintaxe, Vue.filter('nome', funcao(valor){ if(valor){ valor.substring(0,3) + '.' + value.substring(3,6) + '.' + value.substring(6,9) + '-' + value.substring(9); } return ''; })

Para chamar o filtro, usamos o pip (variável | filtro)

Ex:

{{cpf | cpf}}
