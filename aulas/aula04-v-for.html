Entendendo o v-for

A diretiva v-for, simula um loop for no html. Por padrão, o vue no uso de loops, sempre vai precisar de um atributo v-bind:key da diretiva v-bind para que cada elemento tenha uma chave única, tendo assim o gerenciamento das variáveis dentro do laço. Podemos trazer também a chamada do index do array da seguinte maneira.

Ex:
  • Índice: {{index}} -> Posição no array -> {{item.nome}}
